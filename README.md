# Voyager CLI 🚀

A lightweight, persistent Task and Time Management tool built for the terminal.

## 🛠 Features
- **Task Management**: Add, list, and complete tasks.
- **Persistent Storage**: All tasks are saved locally in `tasks.json`.
- **Command-Line Interface**: Built with Python's `argparse` for a native Linux feel.
